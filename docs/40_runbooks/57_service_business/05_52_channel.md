# 4. 52 Channel - продажи сервиса

## Как отметить продажи сервиса в KE30

- На листе KE30 установите признак `1` в столбце **AA** для продаж готовой продукции, относящихся к сервису. Пример: ![Признак сервиса в KE30](6img/KE30_FGSERV.png).
- Можно проставить вручную, либо загрузить данные с фильтром **Канал продаж = 52** на лист **4. 52 Channel** (шаблон: ![Лист 4. 52 Channel](6img/52_ch.png)).
- Формула на листе 4. 52 Channel сверяет два условия: номер фактуры и номер материала.
- После заполнения сравните количество позиций на листах KE30 и 4. 52 Channel, чтобы убедиться, что все продажи сервиса отмечены.
