# 5. KSB1 - затраты сервисных центров

- На листе хранятся затраты, отнесенные на центры затрат департамента сервиса. Шаблон листа: ![KSB1 лист](7img/KSB1.png).
- Просуммируйте данные файла `cc_cost`, который присылает финансовый контролер Лужина И.С. Пример файла: ![cc_cost](7img/CC_costs.png).
- Альтернатива: возьмите данные напрямую из 1C: ERP, выгрузив отчет **F.01** по центрам затрат департамента сервиса.
- После загрузки сверьте, что суммы по центрам совпадают с контрольными показателями в 1C или файле `cc_cost`.
