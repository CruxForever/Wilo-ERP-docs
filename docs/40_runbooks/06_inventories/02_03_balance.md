# 2.3. Данные баланса

Данные баланса используются для сверки запасов с листом BW и последующей консолидации.

![Лист F.01 для переноса баланса](23img/00_F01.png)

## Как получить данные баланса

- Самый быстрый способ: попросить актуальный файл у Лужиной И.С. и перенести значения на лист `F.01` — данные сразу попадут в чек-лист.
- Альтернативно: использовать выгрузку F01, полученную на шаге 1 (см. раздел «Выгрузка данных из 1С»), и заполнить ячейки листа `F.01` вручную.

После заполнения листа `F.01` данные автоматически отразятся на листе `Check`:

![Лист Check с результатами сверки](23img/00_Check.png)
