# 6-Inventories — Обзор

Подготовка листа **6-Inventories** в групповом Excel-пакете — обязательная ежемесячная процедура, обеспечивающая корректную консолидацию данных о запасах. Данные из 1С проходят несколько стадий обработки, проверок и сверок, чтобы итоговые показатели соответствовали требованиям группы и отражали фактическое состояние складов.

## Зачем выполняется процедура

Цель подготовки 6-Inventories — предоставить группе целостную и сопоставимую картину запасов компании:

- **Единый разрез Product Cluster.** Группа анализирует запасы по кластерной структуре, объединяющей номенклатуру на уровне бизнес-логики.
- **Единый поставщик (Supplied By).** Важно видеть, какие запасы привязаны к внешним поставщикам, а какие — внутренние.
- **Разделение по типам запасов (RM / FG / TG).** Нужно для расчётов в SCM, фин. анализе и управления ликвидностью.
- **Сопоставимость между заводами группы.** Формат листа 6-Inventories одинаков для всех предприятий Wilo, поэтому критично соблюдать структуру.
- **Финансовая консистентность.** Объёмы запасов должны соответствовать бухгалтерскому балансу и внутренним отчётам 1С.

**Итог:** корректные показатели в групповом пакете, которые верно попадают в отчётность HQ.
